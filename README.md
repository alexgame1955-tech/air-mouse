# Air Mouse — перчатка-мышь

**Что это:** Перчатка с датчиком. Надеваешь, зажимаешь палец, двигаешь кистью — курсор на экране едет. Отпустил палец — курсор стоит. Ещё есть кнопки для левого и правого клика.

**Как работает:** Подключаешь по USB (кабелем). Компьютер видит как обычную мышь. Никаких драйверов не надо.

---

## 📦 ЧТО НУЖНО КУПИТЬ

| Деталь | Количество |
|---|---|
| ESP32-S3 SuperMini | 1 шт |
| MPU6050 (GY-521) | 1 шт |
| Сенсорная кнопка TTP223 | 3 шт |
| Провода (гибкие, 0.2 мм²) | 3-5 метров |
| Термоусадка (2-3 мм) | по желанию |
| USB кабель Type-C (ДАННЫЕ, не только зарядка!) | 1 шт |
| Перчатка тканевая | 1 шт |

## 💾 КАК ЗАЛИТЬ ПРОШИВКУ (для чайников)

1. **Скачай и установи Arduino IDE** отсюда: `arduino.cc/en/software` (выбирай версию под свою Windows/Mac)

2. **Открой Arduino IDE.** Зайди в `Файл` → `Настройки`. В поле `Дополнительные ссылки` вставь: https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json

3. **Зайди в `Инструменты` → `Плата` → `Менеджер плат`.** В поиске напиши `ESP32`. Найди `esp32 by Espressif Systems` и нажми `Установить`.

4. **Скачай файл прошивки** отсюда: нажми на `air_mouse.ino`, потом на три точки и выбери `Download`.

5. **Подключи ESP32 к компьютеру** кабелем DATA. Зажми и держи кнопку BOOT на плате, потом подключи кабель, потом отпусти BOOT.

6. **В Arduino IDE:** выбери `Инструменты` → `Плата` → `ESP32 Arduino` → `ESP32S3 Dev Module`.

7. **Выбери порт:** `Инструменты` → `Порт` → выбери тот, который появился (например COM3 или COM5).

8. **Открой файл прошивки** в Arduino IDE: `Файл` → `Открыть` → выбери `air_mouse.ino`.

9. **Нажми стрелку вправо (Загрузить)**. Жди, пока напишет `Done uploading`.

10. **Вынь и вставь обратно USB-кабель.** ESP32 перезагрузится, и мышь заработает.

## 🕹️ КАК ПОЛЬЗОВАТЬСЯ

| Кнопка | Действие |
|---|---|
| **GPIO1** | Зажал — курсор едет. Отпустил — стоит |
| **GPIO2** | Нажал — левый клик |
| **GPIO3** | Нажал — правый клик |

---

## ⚙️ КАК ПОМЕНЯТЬ ЧУВСТВИТЕЛЬНОСТЬ

- Открой прошивку (`air_mouse.ino`) в Arduino IDE
- Найди строку: `float sensitivity = 1.5;`
- Поменяй число: 0.5 — медленно, 3.0 — быстро
- Залей прошивку заново

---

## 🧪 ЕСЛИ НЕ РАБОТАЕТ

1. **ESP32 не прошивается?** — кабель только для зарядки. Нужен кабель с поддержкой ДАННЫХ.
2. **Курсор не едет?** — проверь, что кнопка на GPIO1 зажата.
3. **Мышь не видна в Windows?** — в настройках Arduino IDE: `Инструменты` → `USB Stack` → `TinyUSB`, и `USB CDC On Boot` → `Disabled`.
